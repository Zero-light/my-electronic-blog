## 项目概述

在工业测控与实验室环境中，经常需要对多路物理量进行同步采集与远程监控。本项目设计了一套 **智能多通道测量系统**，集成多种传感器，具备本地显示、数据记录与远程上报能力。

## 硬件设计

### 信号调理前端

- **温度**：PT100 + 恒流源激励 + 仪表放大器
- **压力**：MPX5700 压阻式传感器 + 二阶有源低通滤波
- **位移**：LVDT + AD598 信号调理芯片

所有模拟信号经 RC 滤波后送入 ADS1115（16-bit, 860 SPS）进行数字化。

### 通信接口

| 外设 | 协议 | 用途 |
|------|------|------|
| ADS1115 | I2C | 高精度 ADC |
| TFT 屏 | SPI | 本地人机交互 |
| ESP8266 | UART | Wi-Fi 无线传输 |
| SD 卡 | SDIO | 本地数据存储 |

## 软件设计

### FreeRTOS 任务划分

```c
void StartAcquisitionTask(void *argument);
void StartDisplayTask(void *argument);
void StartNetworkTask(void *argument);
void StartStorageTask(void *argument);
```

- **采集任务**：10 Hz 采样率，双缓冲队列投递
- **显示任务**：LVGL 刷新，30 FPS
- **网络任务**：MQTT 发布，1 Hz 心跳
- **存储任务**：FATFS 文件追加，1 分钟写盘一次

### MQTT 数据格式

```json
{
  "device_id": "meas_001",
  "timestamp": 1705312800,
  "channels": [
    {"name": "temperature", "value": 25.6, "unit": "°C"},
    {"name": "pressure", "value": 101.3, "unit": "kPa"},
    {"name": "displacement", "value": 12.5, "unit": "mm"}
  ]
}
```
