## 设计指标

| 参数 | 指标 |
|------|------|
| 输入电压 | 8 V ~ 16 V |
| 输出电压 | 5 V |
| 输出电流 | 3 A |
| 开关频率 | 500 kHz |
| 目标效率 | > 90% |
| 纹波电压 | < 50 mVpp |

## 关键元件计算

### 电感选型

连续导通模式（CCM）下最小电感：

```
L_min = (V_out * (V_in - V_out)) / (V_in * f_sw * ΔI_L)
```

取纹波电流比为 30%：

```
ΔI_L = 0.3 * 3 A = 0.9 A
L_min = (5 * (12 - 5)) / (12 * 500k * 0.9) ≈ 6.5 µH
```

实际选用 **10 µH / 5 A** 屏蔽电感，留足余量。

### 输出电容

```
C_out ≥ ΔI_L / (8 * f_sw * ΔV_out)
C_out ≥ 0.9 / (8 * 500k * 0.05) ≈ 4.5 µF
```

选用 **22 µF × 2（陶瓷）+ 100 µF（电解）**，兼顾 ESR 与容量。

## 环路补偿

Type II 补偿网络设计：

```
R3 = 10 kΩ
C1 = 2.2 nF
C2 = 100 pF
```

交叉频率设定在开关频率的 1/10（50 kHz），相位裕度 > 45°。

![Buck 环路波特图](https://picsum.photos/seed/power2/800/600)

## PCB Layout 要点

1. **开关节点（SW）走线尽量短粗**，降低寄生电感
2. **输入电容紧贴 MOSFET 放置**，减小高频电流环路面积
3. **反馈电阻分压点远离 SW 节点**，必要时用地线隔离
4. **散热过孔阵列** 直接连接到底层铜皮

## 实测数据

| 负载 | 效率 | 纹波 |
|------|------|------|
| 0.5 A | 94.2% | 18 mV |
| 1.5 A | 92.8% | 28 mV |
| 3.0 A | 90.5% | 42 mV |

满载效率达到设计目标，温升 < 35°C（自然对流）。

## 踩坑记录

- **自举电容漏接**：导致高侧 MOSFET 无法完全导通，效率骤降 15%
- **电流采样电阻布局**：开尔文连接必须严格对称，否则环路补偿会引入零点漂移
- **陶瓷电容 DC 偏置效应**：22 µF/16V 在 5V 偏置下实际容量仅剩 8 µF，选型时必须查阅厂商 DC Bias 曲线
